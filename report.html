Website

Design Report

Introduction

This is a proposed new website design for The Garden Indian Restaurant. The aim was to create a modern, responsive and easy-to-use website.

The old website was outdated and not modern. It was harder to read and navigate. This website has been completely redesigned with a modern interface and a focus on clear images and text.

This new website features multiple pages including:

  • Home page
  • Menu pages
  • Online takeaway ordering functionality
  • Reservation form
  • Location page
  • About page

There was a focus on usability, responsiveness and accessibility for all customers.

Design Style

The new design incorporates a warm colour palette to reflect Indian cuisine. The feature text colour is a earthy brown, reminiscent of many of the Indian curries available at the restaurant.

A large hero image at the top of each page is designed to create a strong visual impact and connect the customer with the restaurant's food immediately when they arrive at the website.

Fonts used are simple and clean, and are easy to read. There is one script font also used to add decoration and break up the bold text.

The website has a consistent structure, with the same menu at the top and footer at the bottom.

Navigation

The website can be navigated using the main menu at the top of the site. There are also links in the bottom footer of each page to the menus and back to the home page.

A clear reservation link at the top right enables customers to quickly make a booking for the restaurant.

The website aims to have a clear navigation structure and makes finding everything easy.

Images

While some of the existing supplied images have been used, some require updating and new photos would be highly beneficial.

A number of the provided images were not of high quality or had poor lighting, and their use on the website will detract from the new, modern feel of the site.

The image at the top of every page (with curry, naan bread, onions limes and more) is called the 'hero image'. This image was sourced from a free licence photo website.

The image can be replaced with a high quality image of The Garden or the food, similar to the image now, or the existing image can continue to be used as it is used under a free license.

Other images like the chef on the About page are example images and can be updated with new high quality images of the real chefs at The Garden if desired.

Menus and Online Ordering Form

The takeaway menu has been redesigned into an interactive ordering form where customers can easily place online orders.

Customers can choose their menu sub-section at the top and add or remove quantities for each dish. They can then review the whole order on one page before submitting it.

The customer's last takeaway order can also be reloaded (with customer name and phone) to make repeat ordering fast and easy.

The lunch and dinner menus follow the same format structure just without the ordering functionality. This means the menus are consistent across the whole website.

Bookings/Reservations

A new reservation form has been added to the website so customers can easily submit requests to make bookings.

This form was designed to be simple and easy to use.

The form ensures all the necessary information is included and will not submit without all the correct information.

Mobile Device Compatibility

The website is designed to function across computers, tablets and mobile phones. Pages and layouts will automatically adjust for different devices which helps readability.

Images and menus will also scale appropriately for smaller devices.

Accessibility

The new website provides clear headings and readable text.

Buttons and links are large and easy to interact with.

The website was designed so users of all technological capabilities can easily use the website.

Conclusion

Much thought and care was taken in designing this new website for The Garden. It provides a professional and modern online experience for customers to view menus, order online, make bookings and find out restaurant information.

The new website design also helps improve convenience for customers through online ordering and reservation functionality.

We hope you and your customers enjoy the new website design.
