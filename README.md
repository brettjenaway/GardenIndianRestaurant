# Garden Indian Restaurant Website Redesign

A modern responsive redesign of an outdated restaurant website completed for postgraduate course CSC5740 – Client-Side Web Technology at the University of Southern Queensland.

## Live Website

Garden Indian Restaurant Demo
Hosted on Github pages:
https://brettjenaway.github.io/GardenIndianRestaurant/

## Project Overview

This project involved redesigning and modernising an existing Indian restaurant website while improving:
- responsiveness across devices
- usability and accessibility
- overall visual design
- online takeaway ordering functionality

The redesigned website includes:
- responsive multi-page layout
- dynamic takeaway ordering system
- online reservation form
- interactive menus
- local storage functionality for saving previous takeaway orders

## Features

- Responsive design for desktop, tablet, and mobile devices
- Interactive menu filtering using JavaScript
- Dynamic takeaway order summary and pricing
- Local storage support to remember previous orders
- Online reservation form with validation
- Hamburger navigation menu for mobile devices
- Accessible semantic HTML5 structure
- CSS Grid and Flexbox layouts

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
- CSS Grid
- Flexbox
